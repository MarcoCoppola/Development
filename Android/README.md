### Java packages architecture
```
project
|─ activities
├─ fragments
├─ network
├─ models
├─ managers
├─ utils
└─ views
   ├─ dialogs
   ├─ adapters
   ├─ actionbar
   ├─ widgets
   └─ notifications
```
